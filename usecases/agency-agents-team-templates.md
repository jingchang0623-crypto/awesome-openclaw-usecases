# Agency-Agents: Multi-Agent Team Templates

> A complete AI agency at your fingertips — specialized agent personalities for OpenClaw.

## What it does

From frontend wizards to Reddit community ninjas, from whimsy injectors to reality checkers. Each agent is a specialized expert with personality, processes, and proven deliverables.

Born from a Reddit thread and months of iteration, [agency-agents](https://github.com/msitarzewski/agency-agents) provides meticulously crafted AI agent personalities that you can install directly into OpenClaw.

## Why it's useful

- **🎯 Specialized**: Deep expertise in their domain (not generic prompt templates)
- **🧠 Personality-Driven**: Unique voice, communication style, and approach
- **📋 Deliverable-Focused**: Real code, processes, and measurable outcomes
- **✅ Production-Ready**: Battle-tested workflows and success metrics

Think of it as: Assembling your dream team, except they're AI specialists who never sleep, never complain, and always deliver.

## How to use

### Installation

```bash
# Clone the repository
git clone https://github.com/msitarzewski/agency-agents

# Install all agents to OpenClaw
./scripts/install.sh --tool openclaw
```

### Available Agents

| Agent | Specialty | When to Use |
|-------|-----------|-------------|
| 🎨 Frontend Developer | React/Vue/Angular, UI implementation | Modern web apps, pixel-perfect UIs |
| 🏗️ Backend Architect | API design, database architecture | Server-side systems, microservices |
| 📱 Mobile App Builder | iOS/Android, React Native, Flutter | Native and cross-platform apps |
| 🤖 AI Engineer | ML models, deployment, AI integration | Machine learning features |
| 🚀 DevOps Automator | CI/CD, infrastructure automation | Pipeline development |
| ⚡ Rapid Prototyper | Fast POC development, MVPs | Hackathon projects |
| 🔒 Security Engineer | Threat modeling, secure code review | Application security |

### Activation in OpenClaw

After installation, activate any agent in your OpenClaw sessions:

```
"Hey, activate Frontend Developer mode and help me build a React component"
```

Or specify multiple agents for team coordination:

```
"Have the Backend Architect and AI Engineer collaborate on a ML API design"
```

## Real Example

A one-person company running multiple specialized agents:

1. **Marketing Agent** → Content creation and social media
2. **DevOps Agent** → CI/CD and infrastructure
3. **Security Agent** → Code review and vulnerability assessment
4. **Frontend Agent** → UI development

All coordinated through OpenClaw's multi-agent orchestration.
